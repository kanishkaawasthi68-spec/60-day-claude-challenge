# 🔍 AI Job Red Flag Detector
An intelligent tool that analyzes job descriptions and company info to identify hiring risks, toxic workplace signals, and unrealistic requirements 
before you apply.
## Features
Risk Scoring (0–100) with visual breakdown
Red Flag Detection — burnout culture, missing salary, hustle language
Positive Signal Analysis — genuine green flags in JD
Category-wise Risk Table — 5 categories analyzed
Smart Interview Questions — 5 targeted questions auto-generated
Clean interactive dashboard UI

## Tech Stack
Claude AI (Anthropic) — Core analysis engine
Prompt Engineering — Structured red flag detection logic
HTML / CSS — Interactive risk dashboard UI

## How It Works
1. Paste Job Description + Company Info
2. AI analyzes 5 risk categories
3. Get Risk Score, Red Flags, Positives & Interview Questions
4. Decide: Apply | Caution | Avoid

## Risk Categories Analyzed

| Category | What It Checks |
|---|---|
| Requirements | Experience mismatch, skill overload, contradictions |
| Culture | Burnout signals, hustle language, work-life balance |
| Remote Authenticity | Hidden office clauses, relocation traps |
| Hiring Practices | Missing salary, vague scope, suspicious process |
| Company Stability | Founding year, size, growth or layoff signals |
