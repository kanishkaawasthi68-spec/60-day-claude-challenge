# Prior Authorization Workflow Simulator

## Overview
The Prior Authorization Workflow Simulator is an interactive web application that helps users understand the Prior Authorization (PA) process in the US healthcare system. It provides a step-by-step workflow where users can simulate how a patient case moves through different stages of authorization.

## Features
- Interactive drag-and-drop workflow
- Multiple patient case scenarios
- Patient, Provider, and Payer workflow lanes
- Medical necessity evaluation
- Document collection checklist
- Timeline and workflow progress tracking
- Educational popups explaining each stage
- Different outcomes such as:
  - Approved
  - Denied
  - Pended
  - Appeal Process
- Responsive design for desktop and mobile devices

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## How to Run
1. Clone the repository.
2. Open the project folder.
3. Open `prior-auth-simulator.html` in your browser.

## Learning Objectives
- Understand the Prior Authorization process.
- Learn how providers, patients, and payers interact.
- Explore different workflow outcomes.
- Improve knowledge of healthcare authorization procedures.
