✨ Day 20 - Face Puzzle Game

Implemented a complete Face Puzzle Game using HTML, CSS, and JavaScript.

Features:
- Webcam access using getUserMedia()
- Capture user photo from camera
- Difficulty levels: 3×3, 4×4, and 5×5
- Dynamic puzzle generation from captured image
- Drag & Drop support for desktop
- Touch gesture support for mobile devices
- Timer with live updates
- Move counter and correct piece tracker
- Win detection and result screen
- LocalStorage-based leaderboard (Top 5 scores)
- Retake Photo, Play Again, and New Photo functionality
- Responsive UI for desktop and mobile
- Graceful camera permission error handling

Learnings:
- MediaDevices API (getUserMedia)
- Canvas image processing
- Drag and Touch events
- LocalStorage management
- Game state handling
- Responsive web design
