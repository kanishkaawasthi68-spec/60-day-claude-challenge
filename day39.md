PDF Studio

A modern, responsive PDF utility that allows users to Split and Merge PDF files directly in the browser. All processing happens locally, ensuring complete privacy and fast performance.

✨ Features

Split PDF:

Upload a PDF file
Split PDF using custom page ranges
Split after specific page numbers
Split every N pages
Extract selected pages
Live page thumbnail preview
Zoomable page thumbnails
Download individual split PDFs
Download all split files as ZIP

 Merge PDF:

Merge multiple PDF files into one
Drag & Drop file upload
Reorder PDFs using drag and drop
Preview uploaded files
Download merged PDF instantly



Additional Features:

Drag & Drop support
Keyboard shortcuts
Toast notifications
Loading indicators
Error validation
Accessible UI
Theme switcher


How to Use:

Open the application.
Choose Split or Merge.
Upload PDF file(s).
Select required options.
Click Split PDF or Merge PDFs.
Download the generated PDF(s).
