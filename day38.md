# ⌨️ Typing Speed Studio

Typing Speed Studio is a modern and interactive typing practice web application built using HTML, CSS, and JavaScript. It helps users improve their typing speed, accuracy, and consistency through multiple practice modes and real-time performance tracking.

## 🚀 Features

- ⏱️ Time Mode (15s, 30s, 60s, 120s)
- 🔤 Word Count Mode
- ❝ Quote Typing Mode
- 💻 Programming Practice Mode
  - HTML
  - CSS
  - JavaScript
  - TypeScript
  - Python
  - Java
  - C++
  - SQL
- 📋 Custom Text Practice
- 🧠 Adaptive Mode (difficulty adjusts automatically)
- 🌿 Zen Mode (unlimited typing practice)

## 📊 Live Performance Tracking

- Words Per Minute (WPM)
- Raw WPM
- Accuracy Percentage
- Mistake Counter
- Typing Streak
- Progress Bar
- Remaining/Elapsed Time

## 📈 Detailed Result Analysis

- Final WPM
- Raw WPM
- Accuracy
- Consistency
- Completion Percentage
- Character Statistics
- Typing Rhythm Graph
- WPM Progress Graph
- Accuracy Graph
- Error Heatmap
- Performance Summary
- Achievement Badges

## 🎨 Customization

- Multiple Themes
  - Ink
  - Paper
  - Terminal
- Adjustable Font Size
- Focus Mode
- Sound Effects
- Reduced Motion Option
- Stop on First Error Mode

## 📚 Additional Features

- Session History
- Daily Practice Streak
- Keyboard Shortcuts
- Pause & Resume
- Restart Test
- Responsive Design
- Local Storage Support
- Smooth User Interface


## 🎯 Purpose

The main goal of this project is to help users improve their typing speed, increase typing accuracy, and track their progress through an engaging and user-friendly interface.
