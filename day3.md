# Day 3 - Role Based Prompting
## Question
How can I become a successful web developer?
## Prompt 1 - Without Role
How can I become a successful web developer?
### Learning
General advice was provided.
## Prompt 2 - Founder Persona
You are an experienced startup founder.
How can I become a successful web developer?
### Learning
The response focused on career growth and opportunities.
## Prompt 3 - Developer Persona
You are a senior software developer.
How can I become a successful web developer?
### Learning
The response focused on technical skills and projects.
## Comparison
No Role → General answer
Founder → Business/career perspective
Developer → Technical perspective

## Key Learnings
1. Roles change AI responses.
2. Different personas give different viewpoints.
3. Role-based prompting improves output quality.
