# 🏥 Hospital Admission Readiness Simulator

## 🚀 Features

### 📋 Case Setup
- Provider selection
- Attending physician details
- Diagnosis selection
- Admission type
- Prior Authorization (PA) status
- Admission date

### 📊 Admission Readiness Score

Calculates readiness score based on:

- Prior Authorization
- Documentation
- Physician Orders
- Insurance Verification
- Consent
- Bed Assignment

Displays:

- Readiness Percentage
- Admit Ready
- Near Ready
- In Progress
- Not Ready

---

### 🗂 Prior Authorization Workflow

Supports three scenarios:

- Approved
- Pending
- Denied

For denied cases:

- Review denial reason
- Contact insurance
- Submit appeal
- Appeal success updates workflow automatically

---

### 🏥 Workflow Actions

Interactive workflow including:

- Assign Bed
- Verify Insurance
- Upload Documentation
- Complete Consent
- Contact Physician
- Notify Nursing
- Prepare Patient Arrival

### 📈 Admission Timeline

Visual timeline showing admission progress:

- PA Review
- Insurance Verification
- Bed Assignment
- Documentation
- Consent
- Patient Arrival
- Registration
- Clinical Assessment
- Admission Complete

### ⚠ Risk Tracking

Tracks:

- Documentation Risk
- Insurance Risk
- Bed Risk
- Clinical Risk


### 👨‍⚕ Care Coordination

Displays responsibilities for:

- Attending Physician
- Case Manager
- Nursing
- Utilization Review
- Discharge Planner

### ✅ Final Admission Decision

Shows:

- Admit Ready
- Not Ready

Also displays:

- Missing requirements
- Required actions
- Remaining risks
## 🎯 Project Objective

This simulator is designed to help users understand the hospital admission workflow by simulating:

- Administrative readiness
- Insurance verification
- Prior authorization
- Documentation completion
- Risk assessment
- Admission decision making
