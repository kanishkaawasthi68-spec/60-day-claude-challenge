🚗 Fuel Economics Dashboard
Built a responsive Fuel Economics Dashboard that helps compare different fuel types based on cost, emissions, maintenance, and long-term ownership impact.

## Features:
• Cost per KM analysis
• CO₂ emission comparison
• Fuel type comparison cards
• Vehicle age vs running cost trends
• Maintenance cost insights
• E85 break-even calculations
• Interactive charts & visualizations
• Modern dark-themed dashboard UI
Focused on transforming raw vehicle fuel data into actionable insights through clean visual analytics and an intuitive user experience.
