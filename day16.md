Infosys Stock Fundamental Research Dashboard (FY26)

Built a comprehensive stock research dashboard for Infosys (INFY) using HTML, CSS, and JavaScript. The dashboard presents key financial metrics, valuation insights, revenue trends, cash flow analysis, growth outlook, and investment risks in a clean and interactive format.

Key Features
Financial performance analysis (Revenue, EPS, Profit, Margins)
Valuation metrics (P/E, P/B, Dividend Yield)
Cash Flow & Balance Sheet overview
Growth pipeline and deal wins analysis
Interactive revenue & margin trend chart using Chart.js
Strengths, risks, and investment verdict section
Responsive dashboard design
Key Learnings
Financial data visualization
Dashboard UI design principles
Responsive layouts with CSS Grid
Interactive chart implementation with Chart.js
Structuring complex business data for better readability
