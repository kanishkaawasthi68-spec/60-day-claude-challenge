# 🏥 Prior Authorization Story Simulator

An interactive HTML-based educational simulator that explains the **Prior Authorization (PA)** process in healthcare through a story-driven experience.

## 📖 Overview

This project presents the journey of a patient, **Rahul**, who requires prior authorization for a prescribed medication. Users progress through different scenes to understand how insurance reviews, denials, appeals, and approvals work.

## ✨ Features

- Interactive story-based learning
- Multiple scenes explaining the PA workflow
- User choices that affect dialogue
- Progress tracker
- Responsive UI
- Pure HTML, CSS, and JavaScript
- No external backend required

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Tailwind CSS (CDN)


## 🎯 Learning Objectives

- Understand what Prior Authorization (PA) is.
- Learn how insurance companies review medication requests.
- Explore the appeal process after a denial.
- Understand the complete approval workflow through an interactive experience.
