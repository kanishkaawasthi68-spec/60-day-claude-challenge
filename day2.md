# Day 2 - Prompt Engineering
## Objective
Learned what Prompt Engineering is and how structured prompts improve AI responses.
What I Learned
Prompt Engineering is the skill of writing clear and structured instructions for AI models.
## Benefits:
1. More accurate answers
2. Better structured outputs
3. Faster workflow with fewer corrections
## Weak Prompt
create a website for hotel booking
## Improved Prompt
You're building a hotel booking website. Provide a clear, actionable plan with concrete choices and deliverables. Include:1) Role and scope- Target users (business travelers, families, or general consumers) and primary goals.- Tech stack (frontend framework, language, state management, backend approach, database, and hosting).2) MVP feature set (minimum viable product, 4+ features)- Browsing/search results with filters (location, dates, guests, price, amenities)- Room/detail pages with photos, descriptions, availability calendar, and pricing- Booking flow (guest info, payment method placeholder, receipt)- User accounts (signup/login, saved bookings) optional but recommended3) Non-functional requirements- Performance targets, accessibility, responsive design- Security considerations for payments and personal data4) Architecture and data model- Entity-relationship overview (Hotels, Rooms, Availability, Bookings, Users, Payments, Reviews)- API endpoints (REST or GraphQL) and data flow5) Deliverables and structure- Wireframes or mockups (low or medium fidelity)- Database schema sketch- API contract examples- Frontend component breakdown (Note: include at least HotelList, HotelDetail, BookingWidget, UserAccount)- Deployment plan (staging/production)6) Optional enhancements for post-MVP- Dynamic pricing, reviews, multi-currency, loyalty programs, map integrationPlease confirm constraints (preferred stack, payment gateway, hosting) or adjust as needed, and provide a phased plan from MVP to full feature set.
## Observation
The engineered prompt produced:
Better structure
More detailed explanations
Clear examples
Professional output format
## Key Takeaway
Small improvements in prompts can significantly improve AI output quality.
